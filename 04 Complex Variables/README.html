Hola Mundo

La primer forma de ver "como funciona" un lenguaje es haciendo un hola mundo, ayuda a comprobar que tienes lo minimo necesario para compilar (si es que el lenguaje es compilado) y correr tu programa. El código del programa es así: 

#include 
int main()
{
  std::cout << "Hola Mundo!\n";
}
y para compilar: 
cpp -o hello.cpp
y para ver el resultado: 
./hello
Con esto debes de ver el mensaje del cout en la pantalla.
