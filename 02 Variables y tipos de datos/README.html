Tipos de datos y operadores

Variables

Las variables o identificadores son los elementos de un programa sobre los cuales se realizan acciones, por ejemplo: 

int myVar = 0;

Los identificadores (nombres de variables) solo pueden contener letras (mayúsculas o minúsculas) y el símbolo de guión bajo. El nombre solo puede comenzar con una letra o guión bajo, no puede empezar con un número. Es decir, miVar sería un nombre válido pero 3miVar no. https://isocpp.org/files/papers/N3690.pdf

En su gran mayoría, los tipos de variables con los que vas a trabajar son de tipo numérico, ya sea porque representa una localidad en memoria, un caracter de texto o porque representa un número. Por esta razón vale la pena darle un repaso a su nombre y el rango que van a poder representar.

Y ahora un poco de código para ejercitar un poco lo que dice esta tabla:

#include 

int main()
{
    std::cout << "size of int, - " << sizeof(signed) << "\n";
    std::cout << "size of unsigned int, - " << sizeof(unsigned) << "\n";
    std::cout << "size of __int8, - " << sizeof(char) << "\n";
    std::cout << "size of unsigned __int8, - " << sizeof(unsigned char) << "\n";
    std::cout << "size of __int16, - " << sizeof(short int) << "\n";
    std::cout << "size of unsigned __int16, - " << sizeof(unsigned short) << "\n";
    std::cout << "size of __int32, - " << sizeof(signed) << "\n";
    std::cout << "size of unsigned __int32, - " << sizeof(unsigned) << "\n";
    std::cout << "size of __int64, - " << sizeof(signed long long) << "\n";
    std::cout << "size of unsigned __int64, - " << sizeof(unsigned long long) << "\n";
    std::cout << "size of short, - " << sizeof(signed short int) << "\n";
    std::cout << "size of unsigned short, - " << sizeof(unsigned short int) << "\n";
    std::cout << "size of long, - " << sizeof(signed long int) << "\n";
    std::cout << "size of unsigned long, - " << sizeof(unsigned long int) << "\n";
    std::cout << "size of long long, - " << sizeof(long long) << "\n";
    std::cout << "size of float, - " << sizeof(float) << "\n";
    std::cout << "size of double, - " << sizeof(double) << "\n";
    std::cout << "size of long double, - " << sizeof(long double) << "\n";
}

Aquí la funcion sizeof() se utiliza para calcular el tamaño de la variable en bytes

Tipos con caracteres

En el principio de los tiempos no existían estándares para representar caracteres de texto y uno de los primeros fue ASCII que trabajaba con 8 bits con signo y ahora el mundo de las computadoras está en camino a tener un estándar de 32 bits llamado unicode, éste estándar permite que podamos usar emoticons revueltos con texto normal.

Tipo Bytes Alias Rango
char 1 none –128 a 127 por default 0 a 255 si se compila con /J
signed char 1 none -128 a 127
unsigned char 1 none 0 a 255
wchar_t, char16_t, y char32_t 2 o 4 __wchar_t 0 a 65,535 (wchar_t & char16_t), 0 a 4,294,967,295 (char32_t)
